  <li>Go to <strong>API Keys</strong> tab</li>
  <li>Click <strong>Create API Key</strong></li>
  <li>Copy the key and Workspace ID</li>
</ol>

  <p className="text-yellow-300 m-0">
    <strong>⚠️ Important:</strong> Save your API key now — you won't see it again!
  </p>
</div>

<h2>Step 2: Configure Your Agent</h2>
<p>See the <strong>Setting Up Agents</strong> section for your specific agent (Claude Code, Augment, etc.)</p>

<p>Tell your agent:</p>
<pre><code>"start mai-tai mode"</code></pre>
