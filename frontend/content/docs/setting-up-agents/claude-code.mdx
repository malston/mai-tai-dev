<h1>Configuring Claude Code</h1>
<p className="lead">Connect Claude Code (CLI) to Mai-Tai.</p>

<h2>Setup Command</h2>
<p>Run this in your terminal:</p>
<pre><code className="language-bash">{`claude mcp add mai-tai \
  -e MAI_TAI_API_URL=https://api.mai-tai.dev \
  -e MAI_TAI_API_KEY=mt_your_api_key_here \
  -e MAI_TAI_WORKSPACE_ID=your-workspace-id \
  -- uvx mai-tai-mcp`}</code></pre>

<h2>Start Mai-Tai Mode</h2>
<p>Then tell Claude:</p>
<pre><code>"start mai-tai mode"</code></pre>
