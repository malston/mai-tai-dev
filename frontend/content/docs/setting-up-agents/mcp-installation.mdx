<h1>MCP Server Installation</h1>
<p className="lead">The MCP server connects your AI agent to Mai-Tai.</p>

<h2>Requirements</h2>
<ul>
  <li>Python 3.10+</li>
  <li><code>uv</code> package manager (recommended)</li>
</ul>

<h2>Install with uv</h2>
<pre><code className="language-bash">uv tool install mai-tai-mcp</code></pre>

<p>Or run directly without installing:</p>
<pre><code className="language-bash">uvx mai-tai-mcp</code></pre>

<h2>Alternative: pip</h2>
<pre><code className="language-bash">pip install mai-tai-mcp</code></pre>
