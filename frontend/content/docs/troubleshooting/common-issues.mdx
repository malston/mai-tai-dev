<h1>Common Issues</h1>
<p className="lead">Quick fixes for common problems.</p>

<h2>Agent Not Connecting</h2>
<ul>
  <li>Check your API key is correct</li>
  <li>Verify the Workspace ID matches</li>
  <li>Make sure <code>mai-tai-mcp</code> is installed: <code>uvx mai-tai-mcp</code></li>
  <li>Restart your AI agent</li>
</ul>

<h2>Messages Not Appearing</h2>
<ul>
  <li>Hard refresh the page (Cmd+Shift+R or Ctrl+Shift+R)</li>
  <li>Check your internet connection</li>
  <li>Make sure the agent is in mai-tai mode</li>
</ul>

<h2>Invalid API Key Error</h2>
<p>The API key is incorrect or has been revoked. Generate a new key from workspace settings.</p>
