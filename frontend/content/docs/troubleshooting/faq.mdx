<h1>FAQ</h1>
<p className="lead">Frequently asked questions.</p>

<h2>What agents are supported?</h2>
<p>Claude Code, Augment Agent, and Claude Desktop. Any MCP-compatible agent works.</p>

<h2>Can I use multiple agents?</h2>
<p>Yes! Create separate API keys for each agent. They can connect to the same workspace.</p>

<h2>Is my data secure?</h2>
<p>Yes. All connections use HTTPS/WSS encryption. API keys are hashed and never stored in plain text.</p>

<h2>Can I use Mai-Tai on mobile?</h2>
<p>Yes! Mai-Tai is mobile-first. Open mai-tai.dev on any browser.</p>

<h2>How do I delete a workspace?</h2>
<p>Go to workspace Settings → scroll down → Archive or Delete.</p>
